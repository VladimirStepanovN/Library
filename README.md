# 25.7.1. Практическое задание (PJ-02)

## Задание 25.2.4*

> **Примечание**. Это задание — часть большого итогового задания, которое вы сможете отправить на проверку в конце модуля.

Создайте проект с использованием Entity Framework Core и реализуйте схему Базы данных с помощью классов.
За тему данного проекта возьмите электронную библиотеку, в которой существуют пользователи и книги.
У пользователя обязательно должны присутствовать поля ```Имя``` и ```Email```, а у книги — ```Название``` и ```Год выпуска```.
